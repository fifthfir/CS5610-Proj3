## Instructions to Build & Run
1. Clone the repo
2. 
```
cd client
npm install
```
3. 
```
cd ../server
npm install
```
4. Put the `.env` file under `/server`
5. (Optional) Use `seed.js` to reset and generate new data. It will delete old data.
6. 
```
cd client
npm run dev
```
7. 
```
cd ../server
npm start
```
8. Open: `http://localhost:5173/`
